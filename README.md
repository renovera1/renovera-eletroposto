# Renovera Eletroposto

Landing page publica da Renovera para eletropostos, simulacao comercial, receita projetada, margem e payback.

## Tecnologias

- React
- TypeScript
- Vite

## Como rodar

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

## Publicacao

Este repositorio publica automaticamente no GitHub Pages pelo workflow em `.github/workflows/deploy-pages.yml`.

URL publica:

```txt
https://renovera1.github.io/renovera-eletroposto/
```
